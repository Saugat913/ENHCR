# Nepali Handwritten character Detection with explanation


## Running

####  1. Clone the Project
```bash
git clone https://github.com/Saugat913/ENHCR.git
```
#### 2. Install Requirements
```bash
pip install -r requirements.txt
```
#### 3. Run the project
```bash
flask run
```
